# 畅聊 (PadChat)

专为 vivo Pad5 Pro 大屏平板优化的智能聊天应用。

## 特性

- 📱 **大屏适配** - 横屏双栏布局，充分利用 13 英寸屏幕
- 🤖 **AI 助手** - 内置 GPT-4o / Claude / Kimi 智能对话
- 💬 **即时通讯** - 文字、图片、语音消息
- 🎨 **精美 UI** - Material Design 3 + vivo OriginOS 风格
- 🔒 **隐私安全** - 端到端加密，本地数据保护

## 技术栈

- Flutter 3.24+
- Firebase (认证、数据库、推送)
- Riverpod 状态管理

## 开发

```bash
# 获取依赖
flutter pub get

# 运行
flutter run

# 构建 APK
flutter build apk --release
```
